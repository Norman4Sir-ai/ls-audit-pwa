# LS Audit PWA

**GitHub adres / GitHub URL:**
👉 https://github.com/Norman4Sir-ai/ls-audit-pwa

## Beschrijving

LS Audit is een Progressive Web App (PWA) voor het reconciliëren van Lightspeed POS gegevens. De app laadt een Google Apps Script dashboard in een iframe en kan als app worden geïnstalleerd op telefoon of computer.

## Functies

- 📱 Installeerbaar als PWA op Android en iOS
- 🔄 Service Worker voor offline caching
- 📊 Lightspeed POS audit dashboard via Google Apps Script

## Installatie op telefoon

1. Open https://norman4sir-ai.github.io/ls-audit-pwa in je browser
2. Klik op "Installeer op je telefoon" (Android) of gebruik "Toevoegen aan beginscherm" (iOS)
